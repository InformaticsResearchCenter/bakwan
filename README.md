# bakwan
send messages for whatsapp

# How to use
to using this library by phone number

```python
import bakwan

tempe = bakwan.Bakwan()
tempe.sendNumber("6282217401448", "Hayo bimbingan")
```

to use contact name 

```python
import bakwan

tempe = bakwan.Bakwan()
tempe.sendContact("1184011", "ayo dong bimbingan biar cepet")
```
